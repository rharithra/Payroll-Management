# Salary Management Application

This is a full-stack application for managing employee salary information.

## Technologies Used

### Backend
- Java
- Spring Boot
- MySQL
- JPA/Hibernate

### Frontend
- React
- Bootstrap
- Axios

## Setup Instructions

### Backend Setup

1. Make sure you have Java 17 or higher installed
2. Make sure you have MySQL installed and running
3. Update the database configuration in `src/main/resources/application.properties` if needed
